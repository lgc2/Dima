- Dima.Core - Class Lib (Client/Server)
    - We can't have entity framework here because this lib will be in the client side too

- Dima.Pwa - Blazor WASM (Client)
    - Core

- Dima.Api - API (Server)
    - Core
    - Entity Framwork
